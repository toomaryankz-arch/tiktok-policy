Privacy Policy & Terms of Service

This application uses the TikTok Content Posting API to automate video uploads via the Make.com platform.

Data Collection

We only access video files and captions provided by the user for the purpose of publishing them to their TikTok account. We do not collect or store personal user data.

Data Usage

The data is used exclusively for content synchronization between the user's source and their TikTok profile.

For any questions, please contact the developer via GitHub.
